# Tic Tac Toe

A simple command-line Tic Tac Toe game implemented in Python.

## How to Play

1. Run `python game.py`
2. Players take turns entering numbers 0-8 to place their mark (X or O)
3. First player to get three in a row wins!

## Board Layout
```
0 | 1 | 2
---------
3 | 4 | 5
---------
6 | 7 | 8
```
